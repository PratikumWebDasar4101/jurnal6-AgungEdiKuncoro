Buatlah aplikasi pendaftaran sederhana menggunakan database dengan ketentuan
- Menggunakan session
- Data disimpan ke database
- Terdapat nama (max 35 karakter)
- Nim (min dan max 10 karakter dan harus angka)
- Kelas(radio button)
- Jenis kelamin (radio button)
- Hobi(checkbox)
- Fakultas dropdown
- Alamat textarea
User harus melakukan pendaftaran terlebih dahulu sebelum melakukan login, setelah login aplikasi akan menampilkan data diri user dan user dapat melakukan pengeditan pada data diri
